AngularJS 101

February 11th by Evan

This tutorial will show you the basics of AngularJS and how to get started with it.

Adding Angular to your HTML file

First, you need to start by creating a new html page. Make sure to add your <html>, <head>, and <body> tags.

After creating an index page, you need to insert the code below into your head section.

<script src="http://ajax.googleapis.com/ajax/libs/angularjs/1.2.26/angular.min.js"></script>

What You Should Already Know

Before you start using AngularJS, you should have a basic understanding of:

  • HTML
  • CSS
  • JavaScript

Getting Started

Angular Directives

  • ng-app: Defines the application and tells AngularJS which element is the owner.
  • ng-model: Binds the value of HTML controls to application data.
  • ng-bind: Binds application data to the HTML view.
  • ng-init: Initializes Angular application variables.

Angular Expressions

  • {{Expressions}} are written in brackets and bind data to HTML similar to ng-bind directive.
  • Expressions output data directly where they are written such as below.
<p>This is an example expression: {{5 + 10}}<p>

Angular Controllers

  • Controllers are used to control Angular applications
  • ng-controller defines the controller. Below is an example


Angular Filters

  • Controllers are used to transform data.
  • Filters include: currency, filter, lowercase, orderBy, and uppercase.
  • One example of a helpful filter is currency, as seen below.
<div ng-app="" ng-controller="costController">
	<input type="number" ng-model="quantity">
	<input type="number" ng-model="price">
	<p>Total = {{ (quantity * price) | currency }}</p>
</div>
